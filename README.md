# Software Defect Estimation Using ML Algorithms

This project implements a GUI-based and notebook-based solution to estimate software defects using popular machine learning algorithms on NASA datasets.

## Features

- Random Forest
- Naive Bayes
- Support Vector Machine
- Multilayer Perceptron
- Radial Basis Function
- Bagging
- Accuracy Graph Comparison

## Setup

```bash
pip install -r requirements.txt
```

## Run GUI

```bash
python gui.py
```

## Run Notebook

Open `SoftwareDefectEstimation.ipynb` in Jupyter Notebook.

## Dataset

The dataset is a simplified version of NASA software metrics. Replace `nasa_dataset.csv` with real PROMISE datasets for better accuracy.
