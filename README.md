1.Project Title
Software Defect Estimation using Machine Learning

2. Description
This project predicts whether a software module is defective using machine learning algorithms. It includes data preprocessing, feature engineering, model training, visualization, and a GUI for prediction.

3. Features
Data preprocessing
Machine learning prediction
GUI interface
Accuracy evaluation
Graph visualization

4. Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Tkinter
Jupyter Notebook


5. Future Improvements
Deep Learning
Web deployment (Flask/Streamlit)
More datasets
Cloud deployment
